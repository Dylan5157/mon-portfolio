Motus

Le motus est un jeu qui consiste à trouver un mot d'une certaine taille en un certain nombre de tentatives. Les lettres correctement placées seront affichées aux prochaines tentatives tandis que les lettres présentes seront affichées sans révéler leur réelle position .

Voici ce à quoi peut ressembler un fragment de code du jeu motus, en python :

        def check_word(mot,proposition):

                list_indice= len(mot)*[-1]

                    for k in range(len(mot)):
                        if mot[k] == proposition[k]:
                        list_indice[k]=1
                    for k in range(len(mot)):
                        if list_indice[k] == 1:
                        continue
                    for j in range(len(proposition)):
                        if proposition[j] == mot[k] and list_indice[j] == -1:
                        list_indice[j] = 0
                    break
            
            

        
                return list_indice
