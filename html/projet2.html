Azul

Le azul est un jeu de société qui consiste a remplir un carreau en piochant intelligeamment des tuiles de couleur afin de gagner des points en remplissant le carreau, tout en évitant un maximum les pénalités

Voici ce à quoi peut ressembler un fragment de code du jeu azul, en python :

def start_tour():
    sac = init_game()[0]
    poubelle = init_game()[1]
    cercle_fabrique = init_game()[2]
    
    for k in range(len(cercle_fabrique)):
        for j in range(4):
            if len(sac) <= 0:
                sac = poubelle
                poubelle = []
            shuffle(sac)
            cercle_fabrique[k][j] += sac.pop()
    print(cercle_fabrique)
    print(cercle_fabrique[j])
